Data files

Each file contains in columns, for the model specified;
column 1 time step in years from beginning of the run
column 2 S_N
column 3 S_T
column 4 T_N
column 5 T_T
column 6 q (where available, at 30S)

Salinity (S) and Temperature (T) are averaged over the relevant box volume. _N is the Nor. box, _T is the Trop. box.

Data is given as yearly timeseries.
